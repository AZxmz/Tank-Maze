# Tank-Maze

## 🚀 プロジェクト概要 (项目简介)

`Tank-Maze` は [Unity] と [C#] をベースに開発された3Dタンク迷宮ゲームです。プレイヤーは戦車を操作し、複雑な迷宮の出口を探しながら敵を撃破します。

---

## 🛠️ 使用技術 (技術スタック)

* **ゲームエンジン：** [Unity 2022.3.62f2]
* **プログラミング言語：** [C#], [C++]
* **主要アルゴリズム：** [迷宮生成のためのプリム法]
* **データストレージ：** [Unity PlayerPrefs]
* **開発ツール：** [Visual Studio, Git]

## 🔧 補助ツール：迷宮生成器 (MazeGenerationMaze)

`Tank-Maze` の開発において、迷宮の壁を手動で設計・配置する作業は非常に時間がかかり、単調な繰り返し作業であることに気づきました。この問題を解決するため、迷宮マップを自動生成する補助ツールを開発しました。

### 1. 開発動機
* **効率の向上：** ステージ作成を自動化し、開発時間を「単純作業」から「コアなゲームプレイの設計」にシフトさせます。
* **リプレイ性の向上：** このツールを利用して、ゲーム実行時に動的にステージを生成することが可能です（現在はまだ、人間から見た難易度調整の課題に取り組んでいます）。

### 2. コアアルゴリズム
このジェネレーターは **[プリム法 (Prim's Algorithm)]** を使用しています。
